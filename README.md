# Python酒店管理系统 python23

## 项目概述

本项目是一款基于Python语言的酒店管理系统，旨在实现酒店业务的信息化管理。系统包含前台管理与后台管理两大模块，支持管理员与用户两种角色。

## 技术栈

- 后端：Python + Django框架
- 前端：JQuery + Bootstrap + HTML + JSP
- 数据库：MySQL 5.7.33
- 开发工具：PyCharm

## 功能模块

### 前台管理

- 酒店信息
- 美食信息
- 住宿信息
- 活动信息
- 论坛交流
- 系统公告
- 个人中心
- 后台管理

### 后台管理

- 酒店管理
- 美食信息管理
- 住宿信息管理
- 活动信息管理
- 客房预订管理
- 用户评价管理
- 论坛交流
- 管理员管理
- 系统管理

## 系统角色

- 管理员：负责后台管理，维护系统正常运行。

## 运行环境

- 操作系统：兼容Windows、Linux等平台
- Python版本：3.x（建议3.6以上）
- MySQL版本：5.7.33

## 部署步骤

1. 安装Python环境
2. 安装MySQL数据库
3. 克隆项目代码到本地
4. 导入数据库文件
5. 运行项目
